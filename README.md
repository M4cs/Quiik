# Quiik
Quiik! - A terminal based reaction game.


# Usage

```
usage: quiik [-h] [-rk] [-sm]

optional arguments:
  -h, --help          show this help message and exit
  -rk, --righty-keys  Use IJKL instead of WASD for keys.
  -sm, --skip-menu    Skip Intro
```
